# GeoNarrative

GeoNarrative es una aplicación web diseñada para construir narrativas geoespaciales utilizando datos demográficos, mapas y contenido interactivo.

Este repositorio ha sido reiniciado para comenzar una nueva versión del proyecto.

---

## 🚀 Objetivos del reinicio
- Crear una arquitectura más modular y mantenible  
- Mejorar el flujo de despliegue con Plesk  
- Integrar datos demográficos y GIS de forma más eficiente  
- Optimizar rendimiento y caching  
- Documentar cada etapa del desarrollo

---

## 🧩 Tecnologías
- **ASP.NET Core**
- **JavaScript (ES6)**
- **HTML / CSS**
- **QGIS / GIS**
- **Plesk Obsidian**
- **FTP Simple (VS Code)**
